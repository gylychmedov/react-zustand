## 🖥️ Run Locally

Clone the project

```bash
  git clone https://github.com/gylychmedov/todo-react-zustand.git
```

Go to the project directory

```bash
  cd todo-react-zustand
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
